Ready to Get Started?

Follow these simple steps to install and configure the extension:

  1. Clone the Repository git clone https://github.com/shankerram3/auto-resume-maker-chrome-extension.git
  2. Set Up Backend

    Navigate to backend/, run npm install, add your Anthropic API key to .env, and start with npm start

  3. Load Extension

    Open Chrome ‚Üí Extensions ‚Üí Enable Developer Mode ‚Üí Load unpacked ‚Üí Select extension/ folder

  4. Configure

    Open extension options, set backend URL to http://localhost:3000, and upload your master resume
